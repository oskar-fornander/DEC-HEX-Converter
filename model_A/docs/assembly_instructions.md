# Monteringsanvisningar

Monteringen av DEC-HEX Converter, model A, är tämligen enkel. Komponenterna lödas på kretskortet enligt beskrivningen på pcb:n. Ta hjälp av komponentlistan för att urskilja vilken som är vilken. Batteriboxen limmas fast på botten - observera att du först måste sätta den på on. Pcb:n skruvas fast i höljet, brickorna används för att skruvarna ska passa i längd.
